Dokumentasi API

Autentikasi

Semua request ke API SamudraPay memerlukan API Key yang dikirim melalui header:

X-API-Key: your_api_key_here

Atau menggunakan Bearer Token (untuk user login):

Authorization: Bearer your_jwt_token

Dapatkan API Key dari Dashboard setelah login.

Create Deposit

POST /api/deposit/create

Request Body

Parameter Tipe Required Deskripsi
nominal number Yes Jumlah deposit (minimal 10000)

Response

{ "success": true, "data": { "id": "65a1b2c3d4e5f6...", "nominal": 100000, "fee": 350, "totalBayar": 100350, "metode": "QRIS", "orkutTrxId": "TRX123456", "qrImage": "https://.../qrcode.png", "qrString": "000201010211...", "status": "pending", "expiredAt": "2024-01-13T10:30:00Z", "createdAt": "2024-01-13T10:25:00Z" } }

Contoh Penggunaan (Node.js)

const axios = require('axios'); const response = await axios.post('https://samudrapay.com/api/deposit/create', { nominal: 100000 }, { headers: { 'X-API-Key': 'your_api_key' } }); console.log(response.data);

Cek Status Deposit

POST /api/deposit/status

Request Body

Parameter Tipe Required Deskripsi
depositId string Yes ID deposit dari response create

Response

{ "success": true, "data": { "id": "65a1b2c3d4e5f6...", "nominal": 100000, "fee": 350, "totalBayar": 100350, "status": "success", "expiredAt": "2024-01-13T10:30:00Z", "createdAt": "2024-01-13T10:25:00Z" } }

Status Codes

  • pending - Menunggu pembayaran
  • success - Pembayaran berhasil, saldo bertambah
  • expired - QRIS expired (5 menit)
  • canceled - Dibatalkan user

Cancel Deposit

POST /api/deposit/cancel

Request Body

Parameter Tipe Required Deskripsi
depositId string Yes ID deposit yang akan dibatalkan

Response

{ "success": true, "message": "Deposit canceled successfully" }

Error Handling

Error Response Format

{ "success": false, "message": "Deskripsi error" }

HTTP Status Codes

  • 200 - Success (dengan success: true)
  • 400 - Bad Request (validasi gagal)
  • 401 - Unauthorized (API Key salah/tidak ada)
  • 403 - Forbidden (tidak punya akses)
  • 404 - Not Found
  • 500 - Internal Server Error
